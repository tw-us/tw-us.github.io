# 物資與供應鏈如何進入台灣？

維持供應鏈運作（這需要軍事專家及物流專家）太專業了一般民眾不會接觸到

戰爭初期不會發生斷糧現象，政府會致力維持民眾生存需求，如有徵用民宅及疏散調度之情勢，依照警政局安排即可。但隨著戰爭發展，物資短缺是非常有可能的現象，因此維持供應鏈運作是重要的、需要事先規劃的。
